### Hướng dẫn Cài đặt

  - Tải và cài đặt DNS: Có 3 DNS, ban đầu hãy chọn Config + Install

- Cài đặt app đã ký với cert : Nhớ tên Cert để import sau này
-  Xác minh app: Vào quản lý DNS & VPN, chọn tin cậy và khởi động lại. Vào lại mục này và xác minh app đã cài với chứng chỉ tương ứng (nếu không xác minh được, thử các config của DNS trên)
- Tải thư mục Certificate về
- Giải nén và import cert tương ứng

### Sau khi cài đặt

  - Never disable the Config Profile.
  - Only one Config Profile should be active at a time.
  - Do not sideload too many Apps in short span of time
  - Switching profiles? Enable Airplane Mode first.
  - Before Shutdown/Restart/Update: Enable Airplane Mode + Disable WiFi.
  - Use VPN only after following the VPN Setup Guide.
